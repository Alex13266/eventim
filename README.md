## Eventim

# General Info

# A fully functional eCommerce website where you can purchas tickets for different events
* Front-end: React, SCSS
* Back-end: Firebase

# This project is created using:
* Design System
* Hooks

# Features:
* Home(List products)
* Events
* Halls(coming soon)
* Artists(coming soon)
* Performances(coming soon)
